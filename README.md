# Creative Fusion - Digital Marketing Agency

A comprehensive digital marketing agency platform built with Next.js 16, React 19, and Supabase.

## Features

- **Client Portal**: Book services, manage bookings, track points and wallet
- **Admin Dashboard**: Comprehensive management system for bookings, clients, services
- **Service Catalog**: 14+ service categories with detailed sub-services
- **Portfolio Showcase**: Display client work and case studies
- **Blog System**: Content management and publishing
- **SEO Optimized**: Structured data and meta tags for better search rankings
- **Payment Integration**: Stripe and PayPal support
- **Email Notifications**: Resend integration for transactional emails

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **UI**: React 19, Tailwind CSS v4, shadcn/ui
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Payments**: Stripe, PayPal
- **Email**: Resend
- **Analytics**: Vercel Analytics
- **Deployment**: Vercel

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables (see Vercel dashboard)

3. Run development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000)

## Deployment

Deploy to Vercel with one click or push to your connected GitHub repository.
